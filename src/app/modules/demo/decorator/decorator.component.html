Decorator

Code inside component:

@ngxLocalStorage({key: 'cbox1', prefix: 'demo', nullTransformer: () => false})
boundBoolean: boolean;

@ngxLocalStorage({key: 'txt1', prefix: 'demo'})
boundText: string;

Result:

boundBoolean: {{boundBoolean}}
boundText: {{boundText}}

Using this property decorator will init the property value from the localstorage. The storage key is either resolved by the given key or using the propery name.
